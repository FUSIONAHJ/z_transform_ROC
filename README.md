Overview
Z-Transform ROC Visualizer (Offline) 是一个完全离线运行的 Z 变换收敛域（ROC）互动可视化工具。
用户可以通过拖拽极点、切换序列类型（因果/反因果/双边），实时观察：
• Z 平面中的零极点分布
• 各类序列对应的 ROC 区域
• 单位圆与系统稳定性的关系
• 冲激响应 ℎ[𝑛]的时间域波形
• 收敛域的数学推导与解释
该工具特别适合 信号与系统、数字信号处理（DSP）课程 进行可视化教学，帮助学生直观理解抽象的 Z 变换概念。
🎯 Features（功能介绍）
1. 可交互 Z 平面（Z-Plane Interaction）
• 直接在 Z 平面上拖动极点位置
• 自动计算并展示极点的模与角度
• 实时更新对应的 ROC 与时域响应
• 显示标准元素：坐标轴、网格、单位圆
2. 三类序列的 ROC 动态展示
工具支持三种典型序列结构：
序列类型
ROC 形状
对应颜色
右边序列（因果）
左边序列（反因果）
双边序列
￼
切换序列类型后，ROC 会立即重新计算并显示。
￼
3. 极点管理（Pole Management）
• 一键添加极点
• 删除最后一个极点
• 自动生成合理的坐标初值
• 可将多个极点拖成共轭形式（未强制固定）
• Reset 按钮可恢复默认示例

4. 时间域冲激响应可视化（Time-Domain Plot）
根据极点位置与序列类型，工具自动生成逼真的冲激响应：
• 衰减 / 增幅指数序列
• 震荡响应（复数极点）
• 双边序列响应自动分裂
• 自动裁剪过大数值并标注
• 鼠标悬停可查看精确数值

5. 理论推导与解释（Theory Panel）
工具内置交互式教学内容，包括：
• Z 变换定义式 𝑋(𝑧)=∑𝑥[𝑛]𝑧−𝑛
• 各序列的求和范围说明
• 收敛条件如何推导
• 几何级数收敛解释
• 对应 ROC 数学表达式（动态生成）
内容随极点、序列类型自动变化，是教学与自学的绝佳工具。
6. 系统特性分析（System Analysis）
自动分析系统是否稳定，并说明原因：
• 因果序列：极点必须在单位圆内
• 反因果序列：单位圆必须在 ROC 内
• 双边序列：ROC 必须包含单位圆
显示内容包括：
• ROC 数学式（如∣𝑧∣>0.75）
• 稳定 / 不稳定标识
• 稳定性理由（动态生成）

7. 100% 离线运行（Offline Support）
该工具无需任何外部服务器依赖，全部逻辑通过：
• React 18
• Babel Standalone
• SVG 可视化
• 内嵌 CSS
即可无网环境下独立运行，非常适合课堂演示。

🚀 How to Use
1. 打开 index.html
2. 左侧面板拖动极点或添加新极点
3. 切换序列类型（因果、反因果、双边）
4. 查看实时更新的：
• ROC 区域
• 冲激响应 ℎ[𝑛]
• 收敛域推导
• 系统稳定性
5. 如需恢复初始状态，点击 Reset
